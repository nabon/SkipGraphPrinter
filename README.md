# SkipGraphPrinter
Program for printing the topology of Skip Graph on standard output.
For example, the sample code, which sets the alphabet size of membership vectors as 3, writes out as follows:

```
-*-------------------
-------------------*-

-*-----------------*-
---------*-----------
-----------------*---
-------*-------------
---------------*-----
-------------*-------
-----*---------------
---*-----------------
-----------*---------

-*-----*-----*-----*-
---*-----*-----*-----
-----*-----*-----*---

-*-*-*-*-*-*-*-*-*-*-
```
